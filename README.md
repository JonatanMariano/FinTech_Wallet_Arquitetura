# Mini Projeto - Arquitetura de Software

Projeto acadêmico de arquitetura de software focado na modelagem de uma FinTech Wallet, incluindo definição de requisitos, análise de atributos de qualidade e evolução arquitetural baseada em decisões justificadas e boas práticas da engenharia de software.

## Estrutura

- /docs → documentação geral
- /diagrams → diagramas C4
- /adrs → decisões arquiteturais

## Cenário
FinTech Wallet
